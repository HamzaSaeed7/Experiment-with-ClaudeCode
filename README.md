# Coin Toss

A dark-themed animated coin flip app built with Python and Tkinter.

![Python](https://img.shields.io/badge/Python-3.x-blue) ![Tkinter](https://img.shields.io/badge/GUI-Tkinter-green)

## Features

- Animated coin flip with ease-in-out easing
- Gold (Heads) and silver (Tails) coin visuals with drop shadow and depth effects
- Live stats tracker — Heads, Tails, and total flips
- Dark UI with hover-styled buttons
- Keyboard shortcuts: `Space` or `Enter` to flip

## Requirements

Python 3.x (Tkinter is included in the standard library — no extra installs needed)

## Usage

```bash
python coin_toss_gui.py
```
